# Online Store Database (MySQL)

This project is a MySQL database design for an Online Store.

## Features
- Customers, Products, Orders, Order Items, Payments
- Primary & Foreign key relationships
- Order status tracking
- Sales analysis queries

## Tables
- customers
- products
- orders
- order_items
- payments

## Technologies Used
- MySQL
- MySQL Workbench

## Sample Queries
- Retrieve customer orders
- Calculate total sales
- Find top customers
- Best-selling products
